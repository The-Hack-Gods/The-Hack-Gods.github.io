### **Actives:**

Create Hideout (2 charges) - Grants you one charge of Move Hideout.

Script Kiddie Attack - Select a node, green or white. If the node is white, it has a very low chance to hack the node. Leaves a log.

Denial of Service (2 charges) - Select a green or white node and block all connections to the node this day. Do not leave a log.

Move Hideout (1 charge) - Move your hideout, protecting yourself unless you are occupied.

Wake Up Call (1 charge) - Become the Panicked Blabbermouth.

### **Passives:**

Unsecured Device - If you are arrested, the Feds get some intel, similar to Download Intel, getting intel (see tips for list of intel).

Hidden Connections - If you are murdered, the player that killed you will be arrested.
