### **Actives:**

Unskilled Attack - Select a node, white or green. Leaves a log.

Hack Target (Low) - Select a white node and attempt to hack it. Leaves a log.

Harden Node (D1 -> D3 cooldown, 2 charges) - Select a white node, decreasing hacking ability of getting this node. Strongest hacking ability nerf. Leaves a log.

Hostage Operative (N1 -> N3 cooldown) - Select a target operative and hold them hostage tonight. Choose a night ability the next night to use from the operative’s role for the next phases. Occupy and visit them.

Scout Dead Operative (2 charges) - Select a murdered or arrested operative. Choose an ability from the operative to use from the operative’s role, and use it this phase. Do not visit them.

### **Passives:**

None.
