### **Actives:**

Unskilled Attack - Select a node, white or green. Leaves a log. (You know how to hack, but you don’t want to do it unless absolutely necessary.)

Weaken Connection (2 charges) - Select a target operative, lower their hack ability one stage down permanently.

Sabotage Internet of Things - Select a target operative and lower their hack ability this phase.

Pair Programming (1 charge) - Select a target operative, their hacking ability permanently increases by one stage. Visit them.

Advisor (N1 -> N3 cooldown) - Select a target operative, their hacking ability temporarily increases by one stage. Visit them.

False Advice (N1 -> N3 cooldown) - Select a target operative, their hacking ability temporarily decreases by one stage. Visit them.

Gift Broken Computer (2 charges) - Select a target operative. Permanently grants target operative Very Low hack ability, even if it increases their hack ability. Visit them.

### **Passives:**

Old Guard - If asked the right question, they will receive a Very High hacking ability result.
