### **Actives:**

Hack Target (Moderate, -1 Wanted Level) - Select a white node and hack it, dealing 3 HP. Leaves a log.

Probe Node (1 charge, -1 Wanted Level) - Select a white node and double all damage done to the node.

Exploit Vulnerability (2 charges, -1 Wanted Level) - Select a white node and deal 1 damage more for all hacks to the node.

Ask for Assistance (2 charges, +1 Wanted Level) - Select a target operative and disable all actions related to the topology. Steal their hack chance if it is higher than yours. Works like Jam Network. Visit them.

### **Passives:**

Unsecured Device - If you are arrested, the Feds get some intel, similar to Download Intel, getting intel (see tips for list of intel).

Finding Fame Elsewhere - If you fail to hack 3 consecutive nodes, turn into a Network Shutdowner permanently.
