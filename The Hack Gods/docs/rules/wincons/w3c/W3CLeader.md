### **Actives:**

Take Root Access - Select a target operative. If the target operative has the Root Access, give a chosen operative Root access permanently.

Murder with Goons (3 charges, +1 Wanted Level) - Select an operative and murder them. Cannot be occupied. Occupies the target.

Doxx and Stalk (3 charges) - Select a target operative. Learn this operative's faction alignment.

Cultist Sacrifice - Select a W3C operative and murder him. Can be done alongside any other action.

### **Passives:**

Altar of Heresy - Know if any operative has Root access.

Blood for the Blood God - Murder with Goons occupies the target operative.
