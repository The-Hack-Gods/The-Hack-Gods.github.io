Win Condition: You are given 3 roles to assassinate, and extract an operative afterwards. If a target changes role, you will be notified and must guess that role instead. If a target is killed or arrested, the role will change to another role in the game currently.

### **Actives:**

Investigate - Find out the role of a particular operative. Pierces lies.

Assassinate (+1 Wanted Level) - Select an operative, select a role, and murder them. Pierces and ignores all defensive actions.

Extract - Select an operative, extract them, removing them from the operation. Can only extract your given operatives.

### **Passives:**

None.
