Investigator

Win Condition: You are given 3 roles to find. Find all 3 roles and match them to the operative.

### **Actives:**

Investigate - Find a random piece of information about a selected operative.

Decipher - Select a role to match to the particular operative. Notified if correct and counts towards objective. Can be done in tandem with Investigate.

### **Passives:**

None.
